# TEJIPAZ 3.0

## Visión

Construir una experiencia digital que represente el alma de Tejipaz.

---

## Objetivo

Que un visitante quiera conocer Tejipaz antes de terminar la Home.

---

## Principios

- Menos información.
- Más emoción.
- Menos decoración.
- Más significado.
- Cada página debe sentirse como un capítulo.
- Cada animación debe tener un propósito.
- Cada fotografía debe contar algo.
